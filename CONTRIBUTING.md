# Contributing

Thank you for considering contributing to `laravel-zkteco-adms-server`!

## Getting started

1. Fork the repository.
2. Create a new branch for your feature or bug fix.
3. Run `composer install`.
4. Run `composer test` to verify behavior.
5. Submit a pull request with a clear description of your changes.

## Coding standards

- Use [Laravel Pint](https://laravel.com/docs/pint) for formatting.
- Follow the existing coding style and architecture.
- Keep the package small, maintainable, and consistent with Laravel conventions.

## Reporting issues

If you find a bug or want to request a feature, please create a GitHub issue with:

- a clear description of the problem
- steps to reproduce it
- the expected behavior
- environment information such as PHP, Laravel, and package versions
